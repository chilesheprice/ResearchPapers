# Research Papers Repository

A collection of medical research papers and presentations focusing on women's cardiovascular health and menopause.

## Contents

### Papers
- **Menopause and the Heart.pdf** - Original 2015 research paper by Chileshe Nkonde-Price, MD and Jeffrey R. Bender, MD
- **Menopause_and_the_Heart_2026_Update.md** - 2026 update incorporating latest research and FDA black box warning removal

### Presentations
- **Menopause_and_the_Heart.pptx** - PowerPoint presentation of the 2015 paper
- **Menopause_and_the_Heart_2026_Update.pptx** - PowerPoint presentation of the 2026 update

### Scripts
- **create_presentation.py** - Python script to generate the 2015 presentation
- **create_presentation_2026.py** - Python script to generate the 2026 update presentation

## Key Topics

- Menopause and cardiovascular disease
- Hormone replacement therapy (HRT/MHT)
- The timing hypothesis
- FDA regulatory changes (November 2025 black box warning removal)
- Clinical recommendations for menopausal women
- Risk stratification approaches

## Requirements

To regenerate the PowerPoint presentations:
```bash
pip install python-pptx
python3 create_presentation.py
python3 create_presentation_2026.py
```
